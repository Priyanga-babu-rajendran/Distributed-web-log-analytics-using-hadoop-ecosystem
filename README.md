# 🚀 Distributed Web Log Analytics Using Hadoop Ecosystem
Distributed web log analytics platform for scalable processing and visualization of NASA web server logs.

A scalable distributed analytics platform for processing NASA web server logs using:

- HDFS
- YARN
- MapReduce
- Apache Spark
- Streamlit

## Architecture

NASA Web Logs
→ HDFS Storage
→ MapReduce Processing
→ Apache Spark Analytics
→ Interactive Dashboard

## Features

- HTTP Status Analytics
- Traffic Analytics
- Resource Analytics
- Security Analytics
- Cluster Health Monitoring
- Real-Time Dashboard
