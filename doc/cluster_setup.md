# Hadoop Cluster Configuration

## Cluster Overview

This project uses a distributed Hadoop cluster consisting of:

| Node | Role |
|--------|--------|
| Keerthana | NameNode, ResourceManager |
| Priyanga | DataNode, NodeManager |
| Minha | DataNode, NodeManager |

---

## Hadoop Components

### HDFS

- Distributed file storage
- Data replication
- Fault tolerance

### YARN

- Resource scheduling
- Cluster management

### MapReduce

- HTTP Status Analysis
- Client IP Analysis
- Resource Access Analysis
- Traffic Frequency Analysis

### Apache Spark

- In-memory analytics
- Dashboard data processing
