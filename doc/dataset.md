# NASA Web Log Dataset Documentation

## Dataset Overview

This project uses the NASA Web Server Access Logs as the primary dataset for distributed analytics.

The dataset contains HTTP access records generated by NASA web servers and is commonly used for web log mining, traffic analysis, and distributed computing experiments.

---

## Dataset Source

NASA Kennedy Space Center WWW Server Logs

Original Source:

http://ita.ee.lbl.gov/html/contrib/NASA-HTTP.html

Dataset Used:

NASA_access_log_Jul95.gz

## Log Format

Each record contains:

- Client IP Address
- Timestamp
- HTTP Method
- Requested Resource
- HTTP Status Code
- Response Size

Example:

66.249.73.135 - - [01/Jul/1995:00:00:01 -0400] "GET /images/logo.gif HTTP/1.0" 200 1839

## Analytics Performed

The dataset was analyzed for:

### HTTP Status Distribution

- 200 Success
- 301 Redirect
- 304 Not Modified
- 404 Not Found
- 500 Internal Server Error

### Client Activity Analysis

- Most Active Client IPs
- Request Frequency

### Resource Analysis

- Most Accessed Web Resources
- Asset Type Distribution

### Traffic Analysis

- Hourly Traffic Pattern
- Peak Usage Window
